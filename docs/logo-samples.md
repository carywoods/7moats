# The 7 Moats: Logo Samples

These are first-pass logo directions for review. They use the working tan theme and are intentionally simple SVG marks so the strongest direction can be selected before polish work begins.

## Palette

- Dark background: `#15110c`
- Tan: `#c8aa7a`
- Light tan: `#ead7b1`
- Muted tan: `#8d7249`

## Initial read

The strongest primary candidates are likely **01 Seven Rings**, **02 7M Monogram**, and **10 Open Ring**.

The best full identity system may use:

- **Primary logo:** Seven Rings
- **Small icon / favicon:** 7M Monogram
- **Premium / institutional variant:** Sevenfold Seal

## Review questions

1. Which mark feels most like a serious intellectual framework?
2. Which mark is most memorable at small size?
3. Which mark best fits the phrase, “Human success in the age of intelligent machines”?
4. Which mark would look credible on a Kindle cover, YouTube thumbnail, and workbook PDF?
5. Which mark should be dropped immediately?
