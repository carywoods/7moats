# The 7 Moats

Canonical web home and launch-planning repository for **The 7 Moats**, Dr. Cary Woods' framework for human success in the age of intelligent machines.

## Framework

1. Judgment
2. Trust
3. Craft
4. Adaptation
5. Systems Thinking
6. Ownership
7. Meaning

## Working principle

**Give away the doorway. Sell the room.**

Free public essays, audio, and video build reach and authority. Paid books, workbooks, and discussion guides provide convenience, application, and institutional value.

## Initial site goal

Make `the7moats.com` the canonical framework page that links readers back to Breaking Barriers and the Substack launch essay.
